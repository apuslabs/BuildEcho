# BuildEcho Memory

Use this file to teach BuildEcho the project narrative over time.

## Project

- BuildEcho is an agentic open-source growth company inside every repo.
- It helps open-source projects turn real development progress into public
  proof, build logs, social drafts, community feedback, growth signals, and next
  build steps.
- The first user is an open-source maintainer or solo developer who already
  builds in a Git repository and wants help turning progress into a governed
  daily operating report.

## Public Building Notes

- Share real progress only.
- Emphasize proof, governance, human approval, and useful next build decisions.
- Do not frame BuildEcho as a generic social media bot.
- Do not imply autonomous posting, outreach, or email.
- Tone should be clear, honest, technical, and builder-oriented.
